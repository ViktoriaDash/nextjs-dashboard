Розділ 1-2: 
Реалізація стилів: Підключено глобальні стилі через import '@/app/ui/global.css' у кореневому макеті (RootLayout).
Адаптивність: Впроваджено використання компонента Image для оптимізації LCP.
Responsive Design: Використано класи Tailwind CSS

Розділ 4: 
Root Layout: Створено головний макет додатка, що забезпечує спільні стилі та шрифти для всіх маршрутів.
Dashboard Layout: Реалізовано вкладений макет для авторизованої зони з бічним меню (SideNav)
Маршрутизація: Створено файлову структуру для сторінок /dashboard, /invoices та /customers.

Розділ 7-8: Отримання даних та Dashboard
Data Fetching: Реалізовано отримання даних на стороні сервера для карток статистики, графіків доходу та останніх інвойсів.
Статистика: Налаштовано відображення сум Collected, Pending та загальної кількості інвойсів.

Розділ 12-13: 
Server Actions: Реалізовано функції для створення, оновлення та видалення інвойсів через форми.
Breadcrumbs: Додано навігаційний ланцюжок для зручного переходу між сторінками редагування та списком інвойсів.
Обробка помилок: Додано файл not-found.tsx для обробки випадків, коли інвойс за вказаним ID не знайдено.

Розділ 14:
NextAuth.js: Інтегровано систему аутентифікації для захисту приватних даних.
Middleware: Налаштовано автоматичний редирект
Безпека: Реалізовано перевірку паролів та захист сесій.

Розділ 15: 
Metadata API: Додано об'єкт metadata для кожної сторінки додатка.
Динамічні заголовки: Налаштовано шаблон заголовка у RootLayout
Title Template: Реалізовано статичні метадані для сторінок Login, Invoices, Create Invoice та Customers

Всі дії були підтвердженні скріншотами, які знаходяться у папці /photos

Роздуми, які виникли під час робрти з курсом:

1.Поділ на серверні та клієнтські компоненти спочатку здавався складним. Проте, коли я побачила, як дані з бази даних можна отримувати прямо в async компоненті 
без зайвих useEffect та fetch на клієнті, я зрозуміла перевагу. Це робить код чистішим і безпечнішим.
2.Використання Metadata API замість звичайних тегів <title> у head — це дуже зручно.
